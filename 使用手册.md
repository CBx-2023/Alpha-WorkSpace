# Alpha Workspace 使用手册

欢迎使用 Alpha Workspace！这是一个极简、高效的桌面工作区管理工具。

## 🌟 核心功能

### 1. 自由布局系统
- **拖拽排布**：您可以按住任意图标，将其拖拽到屏幕的任何位置。
- **自动保存**：所有的图标位置变动都会自动保存，下次打开即恢复原样。
- **防止误触**：拖拽后和点击有智能区分，防止操作冲突。

### 2. 收纳桶 (Google Bucket)
- **收纳图标**：将不常用的图标拖拽到右下角的"桶(Bucket)"图标上，即可将其收纳。
- **取出图标**：点击桶图标打开菜单，将里面的图标拖拽回桌面即可取出。
- **内置应用**：桶内默认集成了 AI Studio 和 NotebookLM 快捷入口。

### 3. Typora 深度集成
- **一键启动**：点击 Typora 图标即可快速启动本地 Typora 编辑器。
- **自动配置**：首次运行会自动检测 Typora 安装路径。

### 4. 快捷访问
- **Draw.io**：集成在线绘图工具。
- **Gemini**：快速访问 Google Gemini AI。

## 🎮 操作指南

| 动作 | 说明 |
|------|------|
| **左键单击** | 打开应用或链接 |
| **按住拖动** | 移动图标位置 |
| **拖入桶中** | 收纳图标 |
| **点击桶** | 展开/收起收纳菜单 |
