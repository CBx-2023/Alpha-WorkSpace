# 软件工程设计文档：Alpha Workspace内测版

## 1. 文档概述 (Document Overview)

- **项目名称**：Alpha Workspace内测版
- **版本**：Alpha 内测版 v0.1
- **平台**：Windows 桌面客户端
- **产品定位**：一款极简风格的桌面启动器，旨在整合用户的常用生产力工具（绘图、写作、AI），提供统一的快速访问入口，区分本地应用与云端服务。

------

## 2. 用户界面设计 (UI Specification)

### 2.1 整体视觉风格

- **主题**：深色模式 (Dark Mode)。
- **背景**：深灰色网格背景 (Grid Pattern)，营造工程/极客氛围。
- **布局**：中心化卡片式布局，采用固定边框容器包裹核心功能区。

### 2.2 界面元素布局

- **顶部标签**：
  - 内容：文本 "Alpha内测版"。
  - 样式：悬浮胶囊样式，带有辉光效果，位于主容器上方正中。
- **主容器 (Main Container)**：
  - 屏幕中央的矩形区域，包含三个主要的功能入口。
  - **左上组件**：方形卡片，标签 "draw.io"。
  - **右上组件**：方形卡片，标签 "typora"。
  - **中心组件**：圆角矩形卡片，标签 "gemini"，视觉权重略高于其他组件。
- **右下角功能区 (Bottom-Right Action Area)**：
  - **图标**：Google 风格的桶状图标 (Bucket Icon)。
  - **交互提示**：作为二级菜单的触发入口。

------

## 3. 功能需求说明 (Functional Requirements)

本软件包含 4 个核心交互模块，分别对应不同的系统调用逻辑。

### 3.1 模块 A：在线绘图 (draw.io)

- **UI 标识**：左侧方框 `draw.io`
- **交互逻辑**：单击 (Single Click)
- **系统行为**：
  - 调用系统默认浏览器 (System Default Browser)。
  - 打开目标 URL：`https://app.diagrams.net/` (或用户指定的 draw.io 地址)。
- **异常处理**：若无默认浏览器，弹出系统提示。

### 3.2 模块 B：本地写作 (typora)

- **UI 标识**：右侧方框 `typora`
- **交互逻辑**：单击 (Single Click)
- **系统行为**：
  - 调用 Windows 本地进程。
  - 执行目标路径：查找并启动 `Typora.exe`。
- **配置需求**：
  - 软件初次运行时需检测 Typora 安装路径（或提供设置项供用户手动指定路径 `Path/To/Typora.exe`）。
  - 若未找到路径，应弹出“文件未找到”或“请配置路径”的提示框。

### 3.3 模块 C：AI 助手 (gemini)

- **UI 标识**：中心圆角方框 `gemini`
- **交互逻辑**：单击 (Single Click)
- **系统行为**：
  - 调用系统默认浏览器。
  - 打开目标 URL：`https://gemini.google.com/`。

### 3.4 模块 D：Google 工具集 (Google Bucket)

- **UI 标识**：右下角 Google 桶图标
- **交互逻辑**：
  1. **触发**：鼠标悬停 (Hover) 或 单击 (Click)。
  2. **展开**：在图标上方或旁侧弹出一个**二级菜单 (Sub-menu / Popover)**。
- **子菜单内容**：
  - **选项 1：AI Studio**
    - 行为：调用默认浏览器打开 `https://aistudio.google.com/`。
  - **选项 2：NotebookLM**
    - 行为：调用默认浏览器打开 `https://notebooklm.google.com/`。
- **UI 动效**：点击图标后，建议添加简单的展开动画，体现“从桶中取出”的视觉隐喻。

------

## 4. 技术架构建议 (Technical Architecture)

鉴于该软件主要处理 UI 渲染和简单的系统调用（打开 URL 和 启动 EXE），推荐以下技术栈：

- **方案 A (推荐 - 现代轻量级)**：**Tauri + React/Vue**

  - *优点*：打包体积极小（约几MB），内存占用低，完美支持网格背景和 CSS 动画，Rust 后端处理本地进程调用非常安全高效。
