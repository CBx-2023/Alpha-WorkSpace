# Alpha Workspace 快速开始指南

## ⚠️ 重要提示

在运行此应用之前，您需要先安装 **Rust 开发环境**。

## 安装 Rust（必需）

### 方法 1: 使用 winget（推荐）

打开 PowerShell 或命令提示符，运行：

```powershell
winget install --id Rustlang.Rustup
```

### 方法 2: 官方安装器

1. 访问: https://www.rust-lang.org/tools/install
2. 下载 `rustup-init.exe`
3. 运行安装器，按照提示完成安装
4. 安装完成后，**重启终端**

### 验证安装

重启终端后，运行以下命令验证：

```bash
rustc --version
cargo --version
```

如果显示版本号，说明安装成功！

## 安装 Visual Studio C++ Build Tools（Windows 必需）

Tauri 在 Windows 上需要 C++ 编译工具。

### 方法 1: 使用 winget

```powershell
winget install Microsoft.VisualStudio.2022.BuildTools --override "--wait --add Microsoft.VisualStudio.Workload.VCTools --includeRecommended"
```

### 方法 2: 手动下载

1. 访问: https://visualstudio.microsoft.com/visual-cpp-build-tools/
2. 下载 "Build Tools for Visual Studio"
3. 安装时选择 "Desktop development with C++"

## 运行应用

安装完 Rust 和 C++ Build Tools 后：

### 开发模式（推荐用于测试）

```bash
npm run tauri dev
```

这将启动开发服务器，您可以实时查看应用并测试所有功能。

### 构建可执行文件

```bash
npm run tauri build
```

构建完成后，可执行文件位于：
- `src-tauri\target\release\Alpha Workspace.exe`

双击即可运行！

## 使用说明

1. **draw.io**: 点击左上卡片，在浏览器中打开在线绘图工具
2. **typora**: 点击右上卡片，首次使用需配置 Typora.exe 路径
3. **gemini**: 点击中心卡片，在浏览器中打开 Google AI 助手
4. **Google Bucket**: 点击右下角桶图标，访问 AI Studio 或 NotebookLM

## 故障排除

### 问题: "cargo not found"
**解决**: 需要安装 Rust，请参考上面的安装步骤。

### 问题: 编译错误
**解决**: 确保已安装 Visual Studio C++ Build Tools。

### 问题: Typora 无法启动
**解决**: 检查配置的路径是否正确，例如：
- `C:\Program Files\Typora\Typora.exe`
- 或 `C:/Program Files/Typora/Typora.exe`
